# 📊 Bank Loan Report Dashboard | Excel Analytics Project

## 🚀 Project Overview
This project focuses on building a dynamic and interactive **Bank Loan Analytics Dashboard** using Excel to analyze loan performance, risk distribution, and financial metrics. The dashboard provides insights into loan applications, funding, repayments, and borrower risk segmentation.

---

## 🎯 Problem Statement
Financial institutions often struggle to monitor loan performance and assess risk effectively due to scattered data and lack of real-time insights.

This leads to:
- Difficulty in tracking loan approvals vs defaults  
- Limited visibility into funded vs received amounts  
- Challenges in identifying high-risk (bad) loans  
- Inefficient decision-making in lending strategies  

👉 The goal of this project is to create a centralized dashboard that enables stakeholders to monitor loan performance and make data-driven lending decisions.

---

## 🧠 Objectives
- Analyze total loan applications and funding trends  
- Track loan repayment performance  
- Identify good vs bad loans  
- Evaluate interest rates and debt-to-income (DTI) ratios  
- Provide insights for risk assessment and financial planning  

---

## 🛠️ Tools & Technologies Used
- **Microsoft Excel**
  - Pivot Tables  
  - Pivot Charts  
  - Conditional Formatting  
  - Data Cleaning & Transformation  
  - Dashboard Design  

---

## 🔄 Steps Taken

### 1️⃣ Data Cleaning & Preparation
- Cleaned raw loan dataset by handling missing values  
- Standardized data formats (currency, percentages)  
- Created calculated columns for analysis  

---

### 2️⃣ Data Transformation
- Segmented loans into:
  - Good Loans (Fully Paid)
  - Bad Loans (Charged Off)
  - Current Loans  
- Created derived metrics:
  - Funded Amount  
  - Amount Received  
  - Interest Rate  
  - DTI Ratio  

---

### 3️⃣ Data Analysis
- Analyzed total loan applications and funding trends  
- Compared funded vs received amounts  
- Evaluated loan performance categories  
- Identified risk patterns using DTI and interest rate  

---

### 4️⃣ Dashboard Development
Built an interactive dashboard including:
- KPI cards (Applications, Funded Amount, Amount Received)  
- Good vs Bad loan segmentation  
- Loan performance breakdown (Fully Paid, Charged Off, Current)  
- Interest rate and DTI analysis  
- Visual comparisons across loan categories  

---

## 📊 Key Insights
- Majority of loans fall under **Good Loan category (~86%)**, indicating strong repayment performance  
- Bad loans (~14%) contribute significantly to financial risk  
- Funded amount vs received amount highlights repayment efficiency  
- Higher DTI ratios are associated with increased loan risk  
- Interest rates vary across loan categories, impacting repayment behavior
